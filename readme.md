# 📈 Stock Market Analytics Dashboard

An interactive stock market analytics dashboard built using Python, Streamlit, Pandas, and Yahoo Finance API.

The dashboard allows users to analyze stock performance, market trends, volatility, trading activity, and historical price movements through an intuitive web interface.

## 🚀 Live Demo

https://stock-market-analytics-dashboard-9hdturqgg46f7lblhb6qzb.streamlit.app/

## 💻 GitHub Repository

https://github.com/alvinsaju7479/stock-market-analytics-dashboard

---

## 📌 Features

- Real-time stock data retrieval using Yahoo Finance API
- Analysis of major companies including:
  - Apple (AAPL)
  - Microsoft (MSFT)
  - NVIDIA (NVDA)
  - Tesla (TSLA)
  - Amazon (AMZN)
  - Google (GOOGL)
- Historical stock price analysis
- Daily Returns calculation
- Volatility measurement
- 50-Day Moving Average
- 200-Day Moving Average
- Trading Volume analysis
- Interactive visualizations
- Downloadable CSV datasets
- Web deployment using Streamlit Cloud

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/alvinsaju7479/stock-market-analytics-dashboard.git

cd stock-market-analytics-dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Dashboard

```bash
streamlit run app.py
```

or

```bash
python3 -m streamlit run app.py
```

## 🎯 Project Objectives

- Dashboard Development
- Cloud Deployment
